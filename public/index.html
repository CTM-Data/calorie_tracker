Privacy Policy

Last updated: February 12, 2025

Overview

This is a personal calorie tracking application that uses SMS to log meals. It is built and used solely by one individual for personal health tracking purposes.

Data Collected

The only data collected is the content of SMS messages sent to the application's phone number, which contain descriptions of meals and food consumed. This data is used exclusively to estimate calorie counts and log them to a private spreadsheet.

How Data Is Used

Message contents are sent to an AI API to estimate calorie information, then stored in a private Google Sheet for personal tracking. Data is used solely for the purpose of personal calorie tracking and health monitoring.

Data Sharing

No data is shared with third parties for marketing, advertising, or any other commercial purpose. No data is sold. Message contents are only processed through the AI API strictly for calorie estimation.

Data Retention

Calorie log data is retained in a private spreadsheet indefinitely for personal tracking purposes. The application owner may delete this data at any time.

Contact

For questions about this privacy policy, contact the application owner directly via the registered phone number.
