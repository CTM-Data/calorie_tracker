Program Name

Calorie Tracker SMS

Program Description

Calorie Tracker SMS is a personal calorie tracking service. Users send text messages describing meals or food consumed, and the service responds with estimated calorie counts and a daily running total. This is a personal-use application operated by a single individual.

Message Frequency

Message frequency varies based on usage. Typically, the service sends one reply per incoming message. Users can expect to receive approximately 3 to 10 messages per day depending on how many meals and snacks are logged.

Message and Data Rates

Message and data rates may apply. Please consult your mobile carrier's messaging plan for details on any charges related to sending and receiving text messages.

Opt-Out Instructions

To stop receiving messages from Calorie Tracker SMS, text STOP at any time. You will receive a confirmation message and no further messages will be sent.

Support

For help or questions about the service, text HELP to receive support instructions. You may also contact the application owner directly via the registered phone number.
